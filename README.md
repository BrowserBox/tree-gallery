# Tree Gallery

Public gallery repository for Tree collection scripts.

- Scripts live under `/gallery/collection-scripts/<author>/<script-name>/`
- Build index: `python3 gallery/build_gallery_index.py`
- Validate gallery: `python3 gallery/validate_gallery.py`

Generated artifacts consumed by Grower:

- `gallery/gallery-scripts.json`
- `gallery/gallery-index.json` (compat mirror)
